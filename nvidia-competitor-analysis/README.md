##  How The System Works

```
INPUT → Company Name
   ↓

   ↓

   ↓

   ↓

   ↓
```

---

#  Reusable Prompt System

Below is the exact prompt system used to generate this analysis.

##  MASTER PROMPT

```
Act as a senior business analyst.

Analyze the company: [COMPANY NAME]

Step 1: Provide a structured breakdown including:
- Financial overview
- Strengths
- Weaknesses
- Target audience
- Products/services

Step 2: Identify 3 direct competitors and compare using a table:
Columns: Company, Strengths, Weaknesses, Market Position, Key Offerings

Step 3: Analyze:
- Key risks
- Market gaps
- Strategic opportunities

Step 4: Deliver:
Create a one-page executive summary for decision-makers.
```

 Replace `[COMPANY NAME]` to reuse this system.
