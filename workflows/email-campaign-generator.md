#  Workflow #3 — AI Email Campaign Generator  
## A 4-Step AI System for High-Converting Email Sequences

---

##  Objective

Build a reusable AI workflow that generates a complete email marketing campaign from a simple product or service description.

This system helps businesses:
- Generate leads
- Nurture prospects
- Increase conversions
- Automate communication

---

##  Workflow Overview

This is a **4-step structured AI workflow** executed inside a single conversation.

```
INPUT → Product / Service Description
   ↓
[Step 1] Understand (Audience + Pain Points + Value)
   ↓
[Step 2] Strategy (Email Sequence Plan)
   ↓
[Step 3] Write (Full Email Copy)
   ↓
[Step 4] Optimize (Improve Performance)
   ↓
OUTPUT → High-Converting Email Campaign
```

---

#  Step-by-Step Prompt System

##  Message 1 — Understand

```
Hi Claude,

Act as a senior AI business strategist and marketing consultant.

I am building a portfolio project focused on AI-powered marketing workflows. I want you to help me create a structured email campaign system inside this same conversation.

For Step 1, analyze the following business/service description and identify:
1. The ideal target audience
2. Their top 3 pain points
3. The key value proposition

Present the output in a structured, professional format.

Business Description:

We run an AI-powered Lead Generation & Follow-Up Automation Service for small businesses, agencies, coaches, and local service providers.

Our system captures leads from websites, social media, and messaging platforms. It automatically responds, qualifies leads, sends follow-ups, schedules appointments, and organizes everything in a CRM.

The goal is to improve response time, increase conversions, and eliminate missed opportunities.
```

---

##  Message 2 — Strategy

```
Based on your analysis of the target audience, pain points, and value proposition above,

Design a 5-email nurture campaign.

For each email, define:
1. The goal (awareness, trust, education, urgency, conversion)
2. The subject line approach
3. The core message

Do NOT write full emails yet.

Present the output in a structured table format.
```

---

##  Message 3 — Write

```
Now, using the email sequence strategy above,

Write the full email copy for all 5 emails.

Guidelines:
- Tone: professional, clear, slightly conversational
- Length: 120–180 words per email
- Each email must:
  • Address a specific pain point
  • Deliver value
  • Include a strong call-to-action

Format clearly with:
- Subject line
- Email body
- CTA
```

---

##  Message 4 — Optimize

```
Now optimize the entire email sequence for higher conversions.

Improve:
1. Subject lines (make them more compelling)
2. Opening hooks (more engaging)
3. Clarity and persuasion
4. Call-to-actions (stronger and clearer)

Also suggest:
- 2 A/B test subject lines for each email
- 1 overall improvement strategy for better performance

Present the final version in a clean, structured format.
```

---

#  Workflow Summary

| Step | Purpose | Output |
|-----|--------|--------|
| Understand | Define audience & problem | Audience + Pain Points + Value |
| Strategy | Plan campaign structure | Email sequence framework |
| Write | Generate content | Full email copies |
| Optimize | Improve performance | High-converting campaign |
